# 🌈Color Notes
> 웹 페이지 개발 기간 : 2023.12

### 퍼스널 컬러와 향수 추천을 결합한 맞춤형 웹사이트 프로젝트
사용자의 퍼스널 컬러에 맞는 향수를 추천하고, 탐색 및 구매까지 연결할 수 있는 웹 서비스를 기획하고 UI 중심으로 구현
## 💬의미
- 퍼스널컬러 매칭과 향수의 용어인 노트를 결합하여 ‘Color Notes’ 사이트 제작
- 개인화된 향수 추천 및 구매를 가능

![Logo](https://github.com/ChaeEun0112/WebPage/blob/main/Logo.png)
## 💡개요
- 퍼스널컬러와 향수 매칭을 통해 사용자의 컬러와 어울리는 향수 추천
- 직관적 UI/UX로 쉽게 탐색 및 구매 가능
![Concept](https://github.com/ChaeEun0112/WebPage/blob/main/Concept.jpg)
## 👩‍💻담당 역할
- 웹 페이지 전체 UI/UX 설계
- HTML / CSS 기반 페이지 레이아웃 구현
- 향수 추천 서비스 구조 기획
- 사용자 탐색 흐름 설계
## 🎨제작 과정
### UI/UX 디자인
- HTML, CSS를 활용하여 사용자 친화적인 인터페이스 디자인
- 직관적인 네비게이션으로 쉽게 탐색 가능
### 기능 설계
- 퍼스널컬러별 향수 필터링
- 상세 향수 설명 페이지 이동 
- Q&A 게시판, 사용자 계정 관리 기능로 편의성 향상
